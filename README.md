# poker-rating

## Requirements
- JDK 17. 

## How to check
```shell
./gradlew clean check
```

## How to build
```shell
./gradlew build
```

## How to run rating REST API
```shell
./gradlew :poker-rating:rule-engine:bootRun
```



## Check dependency updates
```shell
./gradlew dependencyUpdates -Drevision=release -DoutputFormatter=html
```
